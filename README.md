# ColorfulEditor
Json editor!

## 介绍

## 库

### 使用的Python库
* pillow==11.1.0
* pygame==2.6.1
* pyinstaller==6.11.1
* pyperclip==1.9.0

### 安装指令
```
python.exe install.py
```

#### 快速安装

在指令后加上

```
-i https://mirrors.aliyun.com/pypi/simple/
```

## 运行

```
python.exe run.py
```

## 打包

```
python.exe build.py
```
